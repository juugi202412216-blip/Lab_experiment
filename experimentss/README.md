# Web Technology Lab Experiments

This repository contains all 10 lab exercises for the Web Technology course (B.Tech Computer Science and Engineering 2023).

## Project Structure

```
lab_experiments/
├── index.html                 # Main landing page with links to all exercises
├── exercise1/                 # Personal Website (Basic HTML)
│   ├── index.html
│   ├── about.html
│   ├── portfolio.html
│   └── contact.html
├── exercise2/                 # E-Commerce Website (Advanced HTML)
│   ├── index.html
│   ├── products.html
│   ├── cart.html
│   └── about.html
├── exercise3/                 # Personal Website with CSS
├── exercise4/                 # E-Commerce Website with CSS
├── exercise5/                 # Calculator with JavaScript
├── exercise6/                 # Forms with JavaScript Validation
├── exercise7/                 # Event Handling
├── exercise8/                 # PHP Form Handling
├── exercise9/                 # PHP & MySQL Integration
└── exercise10/                # Sessions & Cookies
```

## Exercises Overview

1. **Exercise 1**: Personal website with minimum 4 web pages using basic HTML tags
2. **Exercise 2**: Static e-commerce website using advanced HTML tags
3. **Exercise 3**: Enhanced Exercise 1 with Inline, Internal, and External CSS
4. **Exercise 4**: Enhanced Exercise 2 with Inline, Internal, and External CSS
5. **Exercise 5**: Scientific calculator using JavaScript
6. **Exercise 6**: Registration and login forms with JavaScript validation
7. **Exercise 7**: Event handling mechanisms
8. **Exercise 8**: Form handling and validation using PHP
9. **Exercise 9**: Product details management using PHP and MySQL
10. **Exercise 10**: Sessions and cookies implementation

## How to Use

1. Open `index.html` in your web browser
2. Click on any exercise card to view that specific exercise
3. Each exercise is contained in its own folder with all necessary files

## Technologies Used

- HTML5
- CSS3
- JavaScript
- PHP
- MySQL

## Getting Started

Simply open the `index.html` file in any modern web browser to access all lab exercises.

For PHP and MySQL exercises (8-10), you'll need:
- A local server environment (XAMPP, WAMP, or LAMP)
- PHP 7.4 or higher
- MySQL 5.7 or higher
