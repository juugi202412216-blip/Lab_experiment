Welcome to My Website

This is a personal website enhanced with three types of CSS styling: Inline CSS, Internal CSS, and External CSS.

About This Project

This website demonstrates the use of different CSS styling techniques:

📝 Inline CSS

Styles applied directly to HTML elements using the style attribute.

🎨 Internal CSS

Styles defined within the <style> tag in the HTML document head.

📄 External CSS

Styles defined in a separate CSS file (styles.css) and linked to the HTML.

Key Features

  • Responsive Design
  • Modern Gradient Backgrounds
  • Smooth Hover Effects
  • Professional Typography
