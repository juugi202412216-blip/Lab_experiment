## For Exercises 8-10 (PHP & MySQL)

### 🔧 Quick Setup (5 minutes)

#### Windows Users:
```bash
1. Download XAMPP from https://www.apachefriends.org/
2. Install XAMPP
3. Copy lab_experiments folder to C:\xampp\htdocs\
4. Open XAMPP Control Panel
5. Start Apache and MySQL
6. Open browser: http://localhost/lab_experiments/
```

#### Mac Users:
```bash
1. Download MAMP from https://www.mamp.info/
2. Install MAMP
3. Copy lab_experiments folder to /Applications/MAMP/htdocs/
4. Start MAMP servers
5. Open browser: http://localhost:8888/lab_experiments/
```

#### Linux Users:
```bash
sudo apt update
sudo apt install apache2 mysql-server php libapache2-mod-php php-mysql
sudo cp -r lab_experiments /var/www/html/
sudo systemctl start apache2 mysql
# Open browser: http://localhost/lab_experiments/
```

## 🆘 Quick Troubleshooting

### PHP files downloading instead of running?
→ Start Apache server in XAMPP/WAMP/MAMP

### Database connection error?
→ Start MySQL server and check config.php

### Blank page on PHP files?
→ Check PHP error logs in XAMPP/WAMP control panel

### Permission denied (Linux)?
```bash
sudo chmod -R 755 /var/www/html/lab_experiments
```
